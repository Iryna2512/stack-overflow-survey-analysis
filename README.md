# Stack Overflow Survey Analysis

Analysis of the Stack Overflow Developer Survey using **Python (Pandas)**.

## Tools
- Python
- Pandas
- Jupyter Notebook

## Key Tasks
- Calculated total number of respondents
- Analyzed survey response completeness
- Computed mean, median, and mode for WorkExp
- Identified number of remote workers
- Calculated percentage of Python users
- Analyzed learning methods (online courses)
- Computed average and median compensation of Python developers by country
- Identified education level of top 5 highest-paid respondents

## Skills Demonstrated
- Data cleaning and missing value handling
- Descriptive statistics
- GroupBy and aggregation
- Text data filtering
- Percentile calculation
- Multi-condition filtering
